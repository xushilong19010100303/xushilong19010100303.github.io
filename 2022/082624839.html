vue2.0

语法

插值

<body>
  <div id="app">
    {{name}}
  </div>
<script>
    var app = new Vue({
    el : "#app"         //挂载  
    data：{             //必须有data(){return xx}      
        name:'徐世龙' }
})
</script>
</body>
  • .类 #id “div”标签选择器等都可以挂载 不可以挂body html 单标签
  • data里可以任意数据，对象数组都可以

v-？

v-text 修改textContent属性

   <div id="app">
    <h2 v-text = "name"+"!">abc</h2>      // abc被全部替换
    <h2>名字是{{name}}</h2>               //推荐这样写
  </div>

v-html 设置innerHTML属性

    <p v-html = "name"></p>       //显示的是a标签的样式，如果是v-text直接显示原来的内容不会解析
    data:{ name : "<a href='xxx.com'>徐世龙</a>"}

v-on 为元素绑定事件 简写@
事件修饰符@keyup.enter 可传参

     <div id="app">
     <p @click="xsl">点我</p>         //也可以input框设个button        
     </div> 
     <script>
     data:{name='张烁'}
     methods:{ xsl() {alert('cnm'+this.name)} } //方法内部可以通过this访问data的数据
     </script>

v-show 根据真假让元素显示/隐藏 display:none

     <div id="app">
     <p v-show="xsl>18">点我</p>
     </div> 
     <script>
     data:{xsl:20}
     </script>

v-if 操作DOM树 不适合频繁操作的元素，性能消耗大

v-bind 绑定元素的属性 src class title等 省略为：

     <div id="app">
     <img v-bind:src="xsl" :class="{a:isAc}">   //相当于三元表达式isAc？a : ""
     </div> 
     <script>
     data:{xsl:'/abc.jpg' , isAc:false}
     </script>

v-for 生成列表结构

     <div id="app">
     <ul><li v-for = "(a,b) in arr">{{a}}{{b}}</li></ul>
     </div> 
     <script>
     data:{arr:[1,'xsl',3,'^',5]}    //a输出内容。b输出索引
     </script>

不要和v-if联用 因为for优先级高 可以把v-if放在ul或者内部，要不报错。

v-model 双向绑定表单元素的值

     <div id="app">
     <input type="text" v-model="xsl">
     </div> 
     <script>
     data:{xsl:666}                 
     </script>

网络

axios

<body>
<input type="button" value="get请求" class="get">
<input type="button" value="post请求" class="post">
<script src = "https://unpkg.com/axios/dist/axios.min.js"></script>
<script>
    // axios.get().then(function(response){},function(err){})
    document.querySelector(".get").onclick = function()
    {
      axios.get("https://autumnfish.cn/api/joke/list?num=3").then(
        function(response){ console.log(response)},
        function(err){console.log(err)})
    }
    document.querySelector(".post").onclick = function()
    {
      axios.post("https://autumnfish.cn/api/user/reg",{username:"jack"}).then(
         function(response){console.log(response);},
         function(err){console.log(err)} )
    }
</script>
</body>

都是固定写法 背下来吧
axios的this指向改变的问题，先把this存起来(var a = this) 回调函数中使用a

computed计算属性

  • 与data methods同级
  • 最大的特点是缓存，仅当computed内的函数 依赖的变量变化 才会再次运行该函数
  • 本质上其实是get,v-model改变表单会报错

watch

    watch:{
        name(new , old){},  //监听属性
        obj:{handler(new,old){} //监听对象
        deep:true},          
    }

vue3.0

版本

  • Alpha 内测
  • Beta 公开测试 (依然会新增功能，有很多bug)
  • Rc 候选版本 （主要修bug）
  • Release 正式版本

vue3.0新特性

  • ES6 proxy代替ES5 Object.defineProxy 重新实现数据响应
  • 源码用typescript重写
  • 提供了composition API
  • 自定义渲染器 （可以开发App 小程序 游戏开发 Web）
  • Fragment 模板可以有多个元素

数据响应式对比

vue2.0

  • 无法动态监测到对象属性的添加删除
  • 无法监测到数组下标和length属性变更
    解释：vue2.0做一个数据劫持，一旦数据变化，更新DOM
var data{
    age:18;
    name:'张三'
}
for (const key in object) {
    let a = data[key]
   Object.defineProxy(data , k , 
   {get(){
          return a ; 
   }
   set(value){
          a = value;
   }})        
    }

想添加需要$set(对象，属性，值) $delete(对象，属性) arr.push();
vue3.0不支持IE11；
